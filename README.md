# Portfolio Optimization Using Genetic Programming


## genetic programming framework

1. init tree algorithms
2. evaluate tree algorithms
3. make new generation
  - selection
  - subtree crossover
  - mutation
  - replace
4. repeat 2, 3 until satisfying terminal condition
